Streamsx.stargate for HBase on Bluemix

This toolkit is for interacting with HBase via the REST API. It was built to target IBM's Bluemix, which requires that HBase access be via the Knox gateway.

The streamsx.hbase cannot be used in that scenario.

SPL Doc for the functions is available here
